# Data-Analysis-Final-Project
Statistical analysis of insurance, startup, and housing datasets using Hypothesis Testing, Python Regression, and EViews OLS.

## Overview
This project presents a comprehensive statistical analysis of three independent 
datasets sourced from Kaggle.

## Tasks
- **Task 1 — Hypothesis Testing (Excel):** US Medical Insurance dataset (545 obs.)
  — F-test, t-test, Z-test, ANOVA, Tukey-Kramer
- **Task 2 — Multiple Linear Regression (Python):** 50 Startups dataset
  — OLS Regression, R² = 95.07%, R&D Spend as key predictor
- **Task 3 — OLS Regression (EViews):** Housing Prices dataset (545 obs.)
  — R² = 63.27%, Bathrooms & AC as strongest predictors

## Tools Used
- Microsoft Excel
- Python (Pandas, Statsmodels, Sklearn)
- EViews

## Dataset Sources
- Kaggle: US Medical Insurance
- Kaggle: 50 Startups
- Kaggle: Housing Prices
